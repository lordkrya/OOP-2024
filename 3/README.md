# Вариант - Военные учения

## UML диаграмма классов этого проекта провиденав файле: UML_Military-exercises.png

## Реализован шаблонный контейнер: упорядоченная таблица. 

Реализация таблица на векторной структуре: "value_type *arr_;"

Контейнер удовлетворяет условиям стандарта C++17.

Также интератор соблюдает концепт: "std::random_access_iterator<OTableIterator<char, int, true>>"

## Для графики использовалась библиотека SFML

Проект так и на был доведён до конца. 
В данный момент юниты лишь передвигаются в случайном направлении пока не встретят противника,
после завязывается симуляция боя, который представляет из себя простую математическую модель.

Однако большинство классов разных паттернов уже реализованы, 
многие так и не используются во время симуляции. 
Например: погода, тактика (которая есть у экземпляра класса командующего).
